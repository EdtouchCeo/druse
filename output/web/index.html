📝 항목별 기재 방법

1
출결 특기사항

입력 경로: [학급담임] → [학적] → [출결관리] → [출결특기사항등록]

  • 기타결석: 1일이라도 반드시 사유 입력
  • 질병·미인정 결석: 필요시 사유 입력 가능 (개인정보 보호 필요 질병은 기재 금지)
  • 지각·조퇴·결과: 원칙적으로 미입력, 횟수 많을 경우 사유 입력 가능
  • 학생선수 결석: 6시간 누적 = 1일로 간주
  • 개근: 결석·지각·조퇴·결과 1회도 없는 경우 반드시 '개근' 입력
  • 1·2학년: 수업 교시별 정확한 출결 관리 필수 (과목출석률 = 1학점당 16회의 2/3 이상)
2
수상경력
  • 학년 초 학교교육계획에 따른 교내상만 입력 (시상 계획의 수상명과 일치)
  • 수상경력 이외 어떠한 항목에도 교내상 입력 불가
  • 대회 준비과정·참가사실은 수상 여부와 관계없이 어떠한 항목에도 입력 불가
  • '대회' 용어는 수상경력 외 항목에 입력 불가
  • 교과우수상: 교과목 정식 명칭 사용, 여러 과목 수상 시 () 안에 병기
  • 표창장: () 안에 세부 표창 내용 입력
3
창의적 체험활동 — 자율·자치·동아리·진로
  • 기재 가능 범위: ① 학교 주최·주관 국내체험 ② 시도교육감 승인 학교밖기관 ③ 타고교 주관(학교장 승인) ④ 교육관련기관 주관(학교장 승인)
  • 실제로 참여한 시간만 입력 (결석·조퇴로 미참여 시간 제외)
  • 임원 활동: '전교/학년/학급' + 재임기간 () 안에 병기
    예) 학급반장(2026.03.01.-2027.02.28.)
  • 이수시간 0시간 학생: 위탁·장기결석 사유 입력
    예) 위탁학생으로 활동 내용이 없음.
  • 자율동아리: 학년당 1개, 동아리 소개 30자 이내
  • 학교스포츠클럽: 동아리활동에는 클럽명만 입력
  • 1·2학년: 출석인정 결석으로 불참해도 이수시간 인정, 특기사항 미입력
4
창의적 체험활동 — 봉사활동
  • 3학년: 봉사활동 영역으로 시행 (다른 창체 영역 이수시간과 중복 인정 불가)
  • 봉사 동아리 활동은 봉사실적 불인정 (동아리활동으로만 인정)
  • 1·2학년: 창체 각 영역과 연계 운영 가능
    예) 학급자치활동 3시간(학급회의 2시간, 환경캠페인 1시간) → 봉사실적 1시간만 연계
  • 실적연계 사이트(1365, VMS, DOVOL): 인정 가능한 내용인지 반드시 확인 후 반영
  • 1일 최대 8시간 인정: 7교시 수업→1시간 / 6교시 수업→2시간 / 공휴일·휴업일→8시간
  • 학교교육계획 봉사활동: 장소·주관기관명에 '(학교)' 입력
  • 개인봉사활동 실적은 상급학교 진학 시 미제공
5
과목별 세부능력 및 특기사항 (과세특)
  • ★ 모든 교과(군)의 모든 학생을 대상으로 입력 (필수)
  • 교과 미참여 학생: 사유 입력
    장기결석→ "장기결석으로 특이사항 없음."
    위탁학생→ "위탁학생으로 특이사항 없음."
  • 수업 대부분 미참여: 학업성적관리위원회 심의 후 '특이사항 없음' 입력 가능
  • 1학기 교과학습발달상황: 해당 학년도 8월 31일까지 입력
  • PAPS 1등급: 체육 관련 교과 세특에 입력 가능
  • 기재 방향: 교과역량·핵심역량 포함
    (의사소통역량, 자기관리역량, 공동체역량, 추론역량, 태도 및 실천역량 등)
6
독서활동상황
  • 특기할 만한 사항이 있는 학생만 대상
  • 형식: 도서명(저자) 형식으로만 입력
  • 동일 책, 다른 증빙자료 → 중복 입력 가능 (원서·번역본도 각각 가능)
  • 독후활동 외 교육활동 전개 시: 세특·창체 영역에도 도서명 포함하여 입력 가능
  • 독서활동상황은 상급학교 진학 시 미제공
7
행동특성 및 종합의견
  • 학급담임교사가 문장으로 입력 — 추천서 또는 지도 자료로 활용
  • 각 항목에 기록된 자료를 종합하여 작성
  • 누가 기록된 사실에 근거하여 장점·단점 입력
  • 학생의 성장 정도, 특기사항, 발전 가능성 고려하여 교육적 관점으로 작성
  • 부정적 행동특성 입력 시 구체적 누가기록 권장
  • 글자수: 500자 → 300자로 축소
